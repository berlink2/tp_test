## Instructions to run:

 1. Install dependencies with 
 
 >yarn 
 
 or
 
 >npm install

2. To run in development 

> yarn start

or

> npm run start

3. If there are issues with node-sass execute
 
>node node_modules/node-sass/scripts/install.js
